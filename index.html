iCasa IDE Install

Install Eclipse Juno (Eclipse Modeling Tools distribution) on your computer.

  • Run Eclipse and select Help > Install New Software... from the main menu.
  • Click on Add Button and enter a name for the update site (i.e. iCasa Update Site), then enter the update site location http://adeleresearchgroup.github.com/iCasa-IDE/distributions/update-site/, finally click on OK button.
  • Wait for a few seconds until the content of the update site is displayed under combo-box. Select the new update site.
  • Selects all features shown in the list.
  • Continue the standard installation procedure and finally restart your Eclipse to apply changes on the platform.

Setting the iPojo runtime

  1. Download and unzip in your system the iPojo runtime
  2. Run Eclipse and select Window > Preferences ... from the main menu and go to iPojo preferences section.
  3. Click on Browse button and sets the iPojo installation directory using the directory created in step number 1
  4. Finally, click on OK button

Starting the simulator IHM

  • In a new shell, go to the folder where you extract the iPOJO runtime
  • If you're on a Windows OS, run the script startGUI.bat. If you're on a UNIX machine, run the script startGUI.sh
  • If all is correctly set, a message such as "Server started, use Ctrl+D to stop and go back to the console" will be displayed on the prompt

Starting the simulator server

  • In a new shell, go to the folder where you extract the iPOJO runtime
  • If you're on a Windows OS, run the script icasa.teaching.distribution.bat to be in release mode and icasa.teaching.distributionDebug.start to be in debug mode
  • If you're on a UNIX machine, run the script startGateway.sh to be in release mode and debugGateway.sh to be in debug mode
  • Now, everything is set up correctly. Launch your favorite WebBrowser (we encourage to use Firefox) and connect to the address http://localhost:9000/
